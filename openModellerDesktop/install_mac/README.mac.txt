Mac Install Notes:
------------------
- Unzip the application zip (you probably have already done so if 
  you are reading this).
- Drag and drop the openModeller Desktop icon into your applications
  folder in the finder.

Reporting Problems:
-------------------

- For general enquiries use our users mailing list
  
  http://lists.sourceforge.net/mailman/listinfo/openmodeller-users

- For developer related enquiries we have a separate list
  
  http://lists.sourceforge.net/mailman/listinfo/openmodeller-devel

- If you think you have found a bug, please report it using our
  bug tracker at:
  
  http://sourceforge.net/tracker/?group_id=101808&atid=631036

  When reporting bugs, please include some contact information 
  in case we need help with replicating your issue.
